javascript-visualizer

The javascript visualizer (live/code) works on a limited set of javascript statements. It highlights the input's syntax as a tree of gears. This is not it's final form, but a prototype while I get my homework done. In the meantime, simply paste s short snippet of javascript code into the resizeable text area, and hit visualize. The camera could use some work, but keep in mind this is just a prototype.
